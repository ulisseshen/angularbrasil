# Iterable Differ ausente

`NgFor` não conseguiu encontrar um iterable differ para o valor passado. Certifique-se de que é um iterable, como um `Array`.

## Depurando o erro

Ao usar ngFor em um template, você deve usar algum tipo de Iterable, como `Array`, `Set`, `Map`, etc.
Se você está tentando iterar sobre as chaves de um object, você deve consultar o [KeyValue pipe](/api/common/KeyValuePipe).
