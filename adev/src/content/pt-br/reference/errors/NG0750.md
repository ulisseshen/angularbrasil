# Falha ao carregar dependências do @defer

Este erro ocorre quando o carregamento de dependências para um bloco `@defer` falha (normalmente devido a condições ruins de rede) e nenhum bloco `@error` foi configurado para lidar com o estado de falha. Não ter um bloco `@error` neste cenário pode criar uma experiência ruim para o usuário.

## Depurando o erro

Verifique se você adicionou blocos `@error` aos seus blocos `@defer` para lidar com estados de falha.
