# Hydration com Siblings ausentes

Este erro é similar ao [erro NG0500](errors/NG0500), mas indica que o processo de hydration do Angular esperava mais siblings presentes na estrutura DOM em um local específico. Por favor, consulte a página do [erro NG0500](errors/NG0500) para informações adicionais.

## Depurando o erro

Veja o [erro NG0500](errors/NG0500) para mais informações sobre como depurar este erro.
