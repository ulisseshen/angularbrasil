# Suporte JSONP na configuração do HttpClient

Angular produz este erro quando você tenta uma requisição `JSONP` sem fornecer o suporte necessário para isso na configuração do `HttpClient`.
Para habilitar o suporte `JSONP`, você pode fazer uma das seguintes opções:

- adicione o `withJsonpSupport()` como um argumento durante a chamada da função `provideHttpClient` (por exemplo, `provideHttpClient(withJsonpSupport())`) quando `bootstrapApplication` é usadodocs
- importe o `HttpClientJsonpModule` no seu AppModule raiz, quando o bootstrap baseado em NgModule é usado.

## Depurando o erro

Certifique-se de que o suporte JSONP está adicionado à sua aplicação, seja chamando a função `withJsonpSupport` (quando `provideHttpClient` é usado) ou importando o módulo `HttpClientJsonpModule` conforme descrito acima.
