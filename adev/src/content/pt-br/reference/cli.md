# Referência CLI

| Comando                            | Alias      | Descrição                                                                                                      |
| :--------------------------------- | :--------- | :------------------------------------------------------------------------------------------------------------- |
| [`add`](cli/add)                   |            | Adiciona suporte para uma biblioteca externa ao seu projeto.                                                  |
| [`analytics`](cli/analytics)       |            | Configura a coleta de métricas de uso do Angular CLI.                                                         |
| [`build`](cli/build)               | `b`        | Compila uma aplicação ou biblioteca Angular em um diretório de saída chamado dist/ no caminho de saída dado.  |
| [`cache`](cli/cache)               |            | Configura o cache persistente em disco e recupera estatísticas do cache.                                      |
| [`completion`](cli/completion)     |            | Configura o autocompletar do Angular CLI para o seu terminal.                                                 |
| [`config`](cli/config)             |            | Recupera ou define valores de configuração do Angular no arquivo angular.json para o workspace.               |
| [`deploy`](cli/deploy)             |            | Invoca o builder de deploy para um projeto especificado ou para o projeto padrão no workspace.                |
| [`e2e`](cli/e2e)                   | `e`        | Compila e serve uma aplicação Angular, depois executa testes end-to-end.                                      |
| [`extract-i18n`](cli/extract-i18n) |            | Extrai mensagens i18n do código-fonte.                                                                        |
| [`generate`](cli/generate)         | `g`        | Gera e/ou modifica arquivos baseados em um schematic.                                                         |
| [`lint`](cli/lint)                 |            | Executa ferramentas de linting no código da aplicação Angular em uma determinada pasta do projeto.            |
| [`new`](cli/new)                   | `n`        | Cria um novo workspace Angular.                                                                               |
| [`run`](cli/run)                   |            | Executa um target Architect com uma configuração customizada de builder opcional definida no seu projeto.     |
| [`serve`](cli/serve)               | `s`, `dev` | Compila e serve sua aplicação, recompilando quando há mudanças nos arquivos.                                  |
| [`test`](cli/test)                 | `t`        | Executa testes unitários em um projeto.                                                                       |
| [`update`](cli/update)             |            | Atualiza seu workspace e suas dependências. Veja https://angular.dev/update-guide/.                           |
| [`version`](cli/version)           | `v`        | Exibe a versão do Angular CLI.                                                                                |
