# Root element was not found

Components inicializados são definidos na propriedade `bootstrap` de um decorator `@NgModule` ou como o primeiro parâmetro de `bootstrapApplication` para components standalone.

Este erro acontece quando o Angular tenta inicializar um desses components mas não consegue encontrar seu node correspondente no DOM.

## Depurando o erro

Este problema ocorre quando o selector não corresponde à tag

```typescript
@Component({
  selector: 'my-app',
  ...
})
export class AppComponent {}
```

```angular-html
<html>
    <app-root></app-root> <!-- Doesn't match the selector -->
</html>
```

Substitua a tag pela correta:

```angular-html
<html>
    <my-app></my-app> <!-- OK -->
</html>
```
