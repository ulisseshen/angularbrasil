# Alvo de Referência Ausente

<docs-video src="https://www.youtube.com/embed/fUSAg4kp2WQ"/>

O Angular não consegue encontrar uma directive com o nome de exportação `{{ PLACEHOLDER }}`.
Isso é comum com um import ausente ou um [`exportAs`](api/core/Directive#exportAs) ausente em uma directive.

ÚTIL: Este é o equivalente de compilador de um erro de runtime comum [NG0301: Export Not Found](errors/NG0301).

## Depurando o erro

Use o nome da string da exportação não encontrada para rastrear os templates ou módulos usando essa exportação.

Certifique-se de que todas as dependências estão devidamente importadas e declaradas em nossos Modules.
Por exemplo, se a exportação não encontrada for `ngForm`, precisaremos importar `FormsModule` e declará-lo em nossa lista de imports em `*.module.ts` para resolver o erro de exportação ausente.

<docs-code language="typescript">

import { FormsModule } from '@angular/forms';

@NgModule({

imports: [
FormsModule,


</docs-code>

Se você adicionou recentemente um import, precisará reiniciar seu servidor para ver essas mudanças.
