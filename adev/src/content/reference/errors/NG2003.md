# Token Ausente

Não há um injection token para um parâmetro do construtor em tempo de compilação. [InjectionTokens](api/core/InjectionToken) são tokens que podem ser usados em um Dependency Injection Provider.

## Depurando o erro

Observe o parâmetro que lança o erro e todos os usos da classe.
Este erro é comumente lançado quando um construtor define parâmetros com tipos primitivos como `string`, `number`, `boolean` e `Object`.

Use o método `@Injectable` ou o decorator `@Inject` do `@angular/core` para garantir que o type que você está injetando seja reificado \(tenha uma representação em tempo de execução\). Certifique-se de adicionar um provider a este decorator para que você não lance [NG0201: No Provider Found](errors/NG0201).
