# Pipe Não Encontrado

<docs-video src="https://www.youtube.com/embed/maI2u6Sxk9M"/>

O Angular não consegue encontrar um pipe com este nome.

O [pipe](guide/templates/pipes) referenciado no template não foi nomeado ou declarado corretamente.

Para usar o pipe:

- Certifique-se de que o nome usado em um template corresponda ao nome definido no decorator do pipe.
- Adicione o pipe ao array `imports` do seu component ou, se o pipe define `standalone: false`, adicione o `NgModule` ao qual o pipe pertence.

## Depurando o erro

Use o nome do pipe para rastrear onde o pipe é declarado e usado.

Para resolver este erro:

- Se o pipe é local ao `NgModule`, dê a ele um nome único no decorator do pipe e declare-o no `NgModule`.
- Se o pipe é standalone ou é declarado em outro `NgModule`, adicione-o ao campo `imports` do component standalone ou do `NgModule` atual.

Se você adicionou recentemente um import ou declaração, pode precisar reiniciar seu servidor para ver essas mudanças.
