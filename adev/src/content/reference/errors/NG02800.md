# Suporte JSONP na configuração do HttpClient

O Angular produz este erro quando você tenta uma requisição `JSONP` sem fornecer o suporte necessário para ela na configuração do `HttpClient`.
Para habilitar o suporte `JSONP`, você pode fazer um dos seguintes:

- adicionar o `withJsonpSupport()` como um argumento durante a chamada da função `provideHttpClient` (ex.: `provideHttpClient(withJsonpSupport())`) quando `bootstrapApplication` é usado
- importar o `HttpClientJsonpModule` no seu AppModule raiz, quando bootstrap baseado em NgModule é usado.

## Depurando o erro

Certifique-se de que o suporte JSONP foi adicionado à sua aplicação chamando a função `withJsonpSupport` (quando `provideHttpClient` é usado) ou importando o módulo `HttpClientJsonpModule` conforme descrito acima.
