# Export Não Encontrado

<docs-video src="https://www.youtube.com/embed/fUSAg4kp2WQ"/>

O Angular não consegue encontrar uma directive com nome de export `{{ PLACEHOLDER }}`. O nome de export é especificado na propriedade `exportAs` do decorator da directive. Isso é comum ao usar FormsModule ou módulos Material em templates e você esqueceu de importar os módulos correspondentes.

HELPFUL: Este é o equivalente em runtime de um erro comum do compilador [NG8003: No directive found with export](errors/NG8003).

## Depurando o erro

Use o nome de export para rastrear os templates ou módulos usando este export.

Certifique-se de que todas as dependências estão devidamente importadas e declaradas em seus NgModules. Por exemplo, se o export não encontrado é `ngForm`, precisamos importar `FormsModule` e declará-lo na lista de imports em `*.module.ts` para resolver o erro.

<docs-code language="typescript">

import { FormsModule } from '@angular/forms';

@NgModule({
…
imports: [
FormsModule,
…

</docs-code>

Se você adicionou recentemente um import, pode precisar reiniciar seu servidor para ver essas mudanças.
