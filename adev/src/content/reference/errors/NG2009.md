# Seletor Shadow DOM inválido

O seletor de um component usando `ViewEncapsulation.ShadowDom` não corresponde aos requisitos de nome de tag de elemento customizado.

Para que um nome de tag seja considerado um nome de elemento customizado válido, ele precisa:

- Estar em letras minúsculas.
- Conter um hífen.
- Começar com uma letra \(a-z\).

## Depurando o erro

Renomeie o seletor do seu component para que ele corresponda aos requisitos.

**Antes:**

<docs-code language="typescript">

@Component({
selector: 'comp',
encapsulation: ViewEncapsulation.ShadowDom

})

</docs-code>

**Depois:**

<docs-code language="typescript">

@Component({
selector: 'app-comp',
encapsulation: ViewEncapsulation.ShadowDom

})

</docs-code>
