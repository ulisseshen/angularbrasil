# Atributo Inválido

<docs-video src="https://www.youtube.com/embed/wfLkB3RsSJM"/>

Um atributo ou propriedade não pode ser resolvido durante a compilação.

Este erro surge ao tentar fazer binding para uma propriedade que não existe.
Qualquer property binding deve corresponder a:

- Uma propriedade nativa no elemento HTML, ou
- Uma propriedade `input()`/`@Input()` de um component ou directive aplicado ao elemento.

O erro de runtime para isso é `NG0304: '${tagName}' is not a known element: …'`.

## Depurando o erro

Consulte a documentação para a [sintaxe de binding](guide/templates) específica usada. Isso geralmente é um erro de digitação ou import incorreto.
Também pode haver uma directive ausente com seletor de propriedade 'name' ou input ausente.
