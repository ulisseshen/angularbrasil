# Hydration Missing Node

Este erro é similar ao [erro NG0500](errors/NG0500), mas indica que o processo de hydration do Angular esperava que um node estivesse presente na estrutura DOM em um local específico e nenhum foi encontrado. Consulte a página do [erro NG0500](errors/NG0500) para informações adicionais.

## Depurando o erro

Veja o [erro NG0500](errors/NG0500) para mais informações sobre como depurar este erro.
