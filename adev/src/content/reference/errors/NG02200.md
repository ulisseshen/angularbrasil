# Iterable Differ Ausente

`NgFor` não conseguiu encontrar um iterable differ para o valor passado. Certifique-se de que seja um iterable, como um `Array`.

## Depurando o erro

Ao usar ngFor em um template, você deve usar algum tipo de Iterable, como `Array`, `Set`, `Map`, etc.
Se você está tentando iterar sobre as chaves em um objeto, você deveria olhar para o [KeyValue pipe](/api/common/KeyValuePipe) ao invés disso.
