# Elemento Inválido

Um ou mais elementos não podem ser resolvidos durante a compilação porque o elemento não é definido pela especificação HTML, ou não há um component ou directive com tal seletor de elemento.

ÚTIL: Este é o equivalente de compilador de um erro de runtime comum `NG0304: '${tagName}' is not a known element: ...`.

## Depurando o erro

Use o nome do elemento no erro para encontrar o(s) arquivo(s) onde o elemento está sendo usado.

Verifique se o nome e o seletor estão corretos.

Certifique-se de que o component está corretamente importado dentro do seu NgModule ou component standalone, verificando sua presença no campo `imports`. Se o component for declarado em um NgModule (o que significa que não é standalone), certifique-se de que ele é exportado corretamente dele, verificando sua presença no campo `exports`.

Ao usar elementos customizados ou web components, certifique-se de adicionar [`CUSTOM_ELEMENTS_SCHEMA`](api/core/CUSTOM_ELEMENTS_SCHEMA) ao módulo da aplicação.

Se isso não resolver o erro, verifique as libraries importadas para quaisquer mudanças recentes nas exportações e propriedades que você está usando, e reinicie seu servidor.
