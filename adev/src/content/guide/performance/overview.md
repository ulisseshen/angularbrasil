Aprenda sobre diferentes maneiras de otimizar o desempenho da sua aplicação com diferentes estratégias de renderização.
</docs-decorative-header>

Uma das principais prioridades de qualquer desenvolvedor é garantir que sua aplicação seja o mais performática possível. Estes guias estão aqui para ajudá-lo a seguir as melhores práticas para construir aplicações performáticas, aproveitando diferentes estratégias de renderização.

| Tipos de Guias                                        | Descrição                                                                                                                                                                                                      |
| :---------------------------------------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [Renderização server e híbrida](/guide/ssr)          | Aprenda como aproveitar a renderização de páginas no servidor para melhorar os tempos de carregamento.                                                                                                        |
| [Hydration](/guide/hydration)                         | Um processo para melhorar o desempenho da aplicação restaurando seu estado após a renderização server-side e reutilizando a estrutura DOM existente o máximo possível.                                        |
| [Incremental Hydration](/guide/incremental-hydration) | Incremental hydration é um tipo avançado de hydration que pode deixar seções da sua aplicação desidratadas e acionar incrementalmente a hydration dessas seções conforme elas são necessárias. |
